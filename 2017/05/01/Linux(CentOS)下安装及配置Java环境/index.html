下载JDK8

  1. 进入甲骨文官网下载： Java SE Development Kit 8 - Downloads

  2. 选择对应的版本，这里我选择64位Linux版本的压缩包：jdk-8u151-linux-x64.tar.gz

    关于查看当前系统位数，可以使用命令： file /bin/ls 或者 file /sbin/init 查看当前系统的位数。

命令行将会输出以上信息。如果显示 `64-bit` 则为64位，如果显示 `32-bit` 则为32位。
  1. 下载JDK。由于甲骨文官网下载JDK采用基于cookie的防盗链，执着于在linux上下载JDK的同学可以参考 linux命令行下载jdk 这篇文章，但是-=-由于笔者嫌麻烦直接在电脑上下载后通过FTP传到服务器上了
